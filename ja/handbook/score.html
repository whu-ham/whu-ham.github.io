このページの内容

成績 ​

Hamで成績の確認、GPAや総合評価の計算、成績分布の確認ができます。

総合評価の計算タイプ ​

学院によって計算方法が異なります。Hamは共通教育、公共、必修、選択、他学部科目を自動判別し、適切に計算します。

Hamは学院に応じて2つの計算方法を提供します：

B1: 必修・共通教育の成績

B2: 選択・他学部科目の成績

新F2計算方式 ​

計算式：

F2 = B1 + B2 x 0.002
※ B2は最大8科目

この方式では、F2が高くなる8科目を優先的に選択します。

「選択」をタップして、どの選択科目が使われているか確認できます。長押しで任意の成績を選んで計算することも可能です。

特定科目を計算対象から外したい場合は、「成績設定」->「成績有効化の編集」で切り替えられます。

この方式を使用している学院：

  • コンピュータ学院

旧F2計算方式 ​

計算式：

F2 = B1 x 0.98 + B2 x 0.02

この方式を使用している学院：

  • 資源・環境科学学院

JavaScriptで計算 ​

HamはカスタムJavaScriptスクリプトによる計算に対応しています。「成績」->「設定」->「計算方法を選択」で、他のユーザーが共有したスクリプトを選べます。

共有スクリプトの提供元：

  • 内蔵スクリプト
  • Github

単位・GPAの計算 ​

「選択」をタップして選択モードに入り、科目を自由に選んで単位やGPAを計算できます。

単位計算の誤りを避けるため、Hamは複数選択を提供していません。各科目の単位を確認し、学院の計算と一致することを確認してください。

現在は複数選択に対応していますが、単位の確認は引き続き必要です。

セキュリティと生体認証 ​

成績を他人に見られないよう、デフォルトで生体認証が有効です。成績を表示するにはFace IDや指紋認証などが必要です。

不要な場合は設定で無効にできます。

成績分布 ​

Hamで授業の成績分布を確認できます。

利用前に必要なこと ​

成績分布を使うには：

  • 成績を取得済みであること
  • 匿名成績をHamサーバーに送信することに同意すること

送信データは匿名化されます。学籍番号は直接送られず、特徴量からハッシュ化されます。サーバー上で個人を特定できません。

「成績分布」を開くたびに、Hamは成績をサーバーに送信し、他のユーザーに参考情報を提供します。

送信を拒否することもできますが、その場合は成績分布機能は利用できません。

分布の検索 ​

検索ボックスに科目名や教員名を入力すると、特定の科目または教員の分布を検索できます。

科目名は完全に入力する必要はありません。例えば「毛泽东思想和中国特色社会主义理论体系概论」は「毛概」でも検索できます。

授業レビュー ​

受講した授業に匿名レビューを投稿できます。投稿後、他の学生がその授業を検索するとコメントと評価が表示されます。

レビューを投稿した後に他のレビューを閲覧できます。

受講したい ​

興味のある授業は詳細ページで「受講したい」をタップしてください。

必要な権限 ​

生体認証 ​

  • iOS: Face ID
  • Android: 端末のパスコード有効化（権限ではないが必須）
