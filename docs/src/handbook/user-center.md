# 用户中心

登录Ham账号后，你可以在用户中心管理你的账号信息和相关设置。

## Ham账号

Ham支持以下登录方式：
- Passkey登录
- QQ登录
- 微信登录
- Github登录
- 自强Auth登录
- Apple登录（仅iOS）

## 个人信息

登录后，你可以在用户中心修改你的头像和昵称。

## 登录设备

你可以查看当前已登录的设备列表，也可以移除不再使用的设备。

## 社交账号

你可以查看已绑定的社交账号，也可以绑定新的社交账号或解绑已有的社交账号。支持绑定的社交账号类型与登录方式一致。

## Passkey管理

你可以在此管理你的Passkey，包括添加新的Passkey或移除已有的Passkey。Passkey是一种无密码的登录方式，使用设备的生物识别（如Face ID、指纹）进行身份验证。

## 扫码登录

你可以使用App扫描Ham Web上的登录二维码，快速完成登录。

## 授权应用

你可以查看和管理通过Ham账号授权登录的第三方应用。如果你不再使用某个应用，可以撤销其授权。

## 更多

在"更多"中，你可以进行账号注销等操作。
