# 日程
在日程页中，你可以查看所有日程、添加或删除日程。

## 最近日程
进入日程页后，页面顶部会展示距离当前时间最近的一条未过期日程，包括日程名称、时间、地点，以及距离开始的倒计时。如果该日程关联了课程，还会显示关联的课程名称。

点击最近日程卡片可以查看该日程的详细信息。

## 日程群组
你可以创建群组来分类管理日程。群组显示在日程列表上方，点击群组可以筛选该群组下的日程。每个群组可以设置名称和图标。

## 添加日程
点击右上角的"+"按钮添加新日程。添加日程时，你可以设置以下信息：

- **名称**：日程的标题（必填）
- **地点**：日程的地点
- **备注**：日程的详细内容
- **目标时间**：日程的开始时间
- **结束时间**：可选，日程的结束时间（与开始时间相隔不能超过24小时）
- **群组**：将日程归入某个群组
- **提醒时间**：在日程开始前提醒你，可选择10分钟、30分钟、1小时、2小时、4小时、8小时
- **重复**：设置日程的重复规则，支持按天、按周、按月重复，并可设置重复结束日期
- **关联课程**：将日程与课程表中的某门课程关联

## 编辑与删除日程
点击日程列表中的某条日程，可以查看其详细信息。在详情弹窗中，你可以：
- 点击设置按钮进入编辑页面，修改日程信息
- 点击删除按钮删除该条日程
- 如果日程设置了重复，可以选择删除所有关联的重复日程

## 系统日历集成（iOS）
添加或编辑日程后，Ham会自动将日程同步到系统日历中（日历名称为"Ham-日程"）。日程的名称、地点、备注、时间和提醒信息都会同步到系统日历。

## 通知提醒
如果你设置了提醒时间，Ham会在日程开始前通过系统通知提醒你。通知内容包括日程名称、地点和开始时间。

## 必要的权限
### 日程提醒
- iOS: 通知、日历
- Android: 通知权限
