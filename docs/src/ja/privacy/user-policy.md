# 利用規約

本規約は、Hamとユーザーの間におけるHamソフトウェアサービス（以下「本サービス」）の権利義務関係を定めるものです。「ユーザー」とは、本サービスをダウンロード、インストール、登録、ログイン、利用（以下総称して「利用」）する個人を指します。Hamは、登録前に本利用規約（以下「本規約」）をよく読み、内容を十分理解するようご注意ください。特に、責任の免除・制限、権利許諾、情報利用、特別サービスの同意、準拠法および紛争解決に関する条項は重要です。重要な免責・制限条項は強調表示されます。これらに同意できない場合、本サービスの利用を中止してください。本サービスを利用した場合、本規約および関連する規約・ルール（本規約を含む）に同意したものとみなされます。

より良いサービス提供のため、または法令、政策、技術、機能の変更により、Hamは本規約を改定することがあります。改定内容は本規約の一部を構成します。更新後は公式サイトの告知など適切な方法で通知します。ユーザーは公式サイトまたはアプリ設定で最新版を確認できます。更新後に本サービスを継続利用する場合、改定後の規約に同意したものとみなされます。

## 1. アカウント登録

1. 本サービスの利用にはHamアカウントの登録が必要です。登録・連携方法は、ユーザーニーズや製品要件に応じて変更される場合があります。

2. アカウント連携の都合上、登録時に提供されたソーシャルアカウントや電話番号等をHamが利用することに同意します。

3. アカウント名、アバター、プロフィール等の登録情報は法令およびHamの規則に従う必要があります。違法・不適切な情報を含めてはならず、他人を装う行為（氏名やアバターの冒用等）や混同を生じさせる行為は禁止されます。悪意のある登録（頻繁・大量登録等）も禁止されます。Hamは登録情報を審査する権利を有します。

4. 登録や利用の過程で、必要に応じて個人情報を収集することがあります。氏名、性別、年齢、生年月日、身分証番号、住所、学校、会社、業界、趣味、行動範囲、自己紹介などが含まれる場合があります。これらの利用は第3条の個人情報保護に従います。

## 2. サービス内容

1. 本サービスの具体的な内容はHamが実際の状況に応じて提供します。星座運勢、天気などを含む場合があります。ユーザー体験向上のため、Hamはサービス内容を随時変更することがあります（ソフトウェア改修、アップグレード、機能強化、新機能の追加、置換等）。変更時は通知します。更新を選択しない場合、一部機能が制限されることがあります。

2. Hamはユーザーに対し、個人的、取消可能、譲渡不可、非独占的、非商業目的の合法な使用権を付与します。本規約で明示されていない権利はHamに留保されます。行使にはHamの書面による許諾が必要です。権利を行使しないことは放棄を意味しません。

3. Hamは無料サービスと有料サービスを提供します。有料サービスの支払い方法は、オンラインバンキング、Alipay、またはHamが提供するその他の方法です。有料サービスは利用前に明確な提示を行い、ユーザーが同意して支払いを完了した場合にのみ利用できます。支払い完了は銀行または第三者決済プラットフォームの「支払い完了」通知をもって確認します。

## 3. ユーザー個人情報の保護

1. 登録や利用の過程で、法令により必要な個人情報の提出が求められる場合があります。情報が不完全または不適合な場合、本サービスを利用できない、または利用が制限されることがあります。

2. ユーザー個人情報には、(1) ユーザーが提供する情報（電話番号、位置情報など）、(2) 他者から共有された情報、(3) サービス提供のために合法的に収集される情報（端末情報、閲覧履歴、時間情報等）が含まれます。個人情報とは、単独または他の情報と組み合わせて個人を特定できる情報を指します。Hamは公開するプライバシーポリシーに従って個人情報を処理します。全文を確認し、適切な判断を行うことを推奨します。

3. Hamは個人情報の保護を重視し、技術的その他の必要な措置を講じて安全性を確保します。漏えい、毀損、紛失が発生した場合、または発生の可能性がある場合、Hamは速やかに対応し、ユーザーへ通知します。ユーザーがそのような事象を発見した場合、直ちにHamへ連絡してください。

4. Hamはユーザーの同意なく個人情報を第三者に開示しません。ただし、以下の場合は除きます。
    - 法令や権限機関の指示に基づく場合
    - ユーザーがパスワードを共有するなどユーザー側の要因により漏えいした場合
    - ユーザー自身が第三者に公開した場合
    - ユーザーとHamまたは協力先が個人情報の利用に合意し、それに基づき開示する場合
    - ハッキング、ウイルス、不可抗力による漏えいの場合
    - 個人を特定できないよう処理され、復元できない場合

5. Hamは以下の目的で個人情報を利用します。
    - ソフトウェア更新や規約変更など重要な通知の送信
    - 監査、データ分析、研究による製品改善
    - 本規約に基づくユーザー情報の管理・審査・処理
    - 法令で定められたその他の事項

上記以外の用途には、事前の同意なく利用しません。

6. Hamは未成年者の保護を重視します。ユーザー提供情報により未成年かどうかを判断し、18歳未満の未成年者は登録または利用できません。

7. Hamは合法・正当・必要な範囲で個人情報を収集・利用・公開し、サービスと無関係な個人情報は収集しません。

8. Hamはプライバシーポリシーを制定しており、ユーザーはその内容を理解したうえで利用する必要があります。同意しない場合、サービスが正常に動作しない可能性があります。設定の「概要」からも確認できます。

1. 本条における「コンテンツ」とは、ユーザーが本サービス利用中に作成・投稿・複製・公開・配信する一切の内容を指します。アカウントのアバター、名称、紹介文、認証情報、文字、音声、画像、動画、図文、リンクなどが含まれます。

2. ユーザーは、以下の法令・政策で禁止される内容を作成・投稿・複製・公開・配信してはなりません。

    - 国家安全保障の侵害、国家秘密の漏えい、政権の転覆、国家統一の破壊

    - 英雄烈士の事績や精神を歪曲・中傷・否定し、名誉等を侵害する内容

    - 民族差別や団結の破壊
    - 宗教政策の破壊、カルトや迷信の宣伝
    - デマの流布、経済・社会秩序の混乱
    - わいせつ、賭博、暴力、殺人、テロ、犯罪教唆
    - 侮辱や名誉毀損、プライバシー侵害
    - 違法な集会・結社・デモの扇動
    - 非合法団体の名義での活動
    - 法令、社会制度、国家利益、公民の合法的権益、公共秩序、道徳、情報の真実性の「7つの底線」に違反する内容

3. ユーザーは、Hamの正常な運営を妨げたり、他のユーザーまたは第三者の権利を侵害する内容を作成・投稿・複製・公開・配信してはなりません。具体例：
    - 誇張された見出しで内容と一致しないもの
    - ゴシップやスキャンダルの煽動
    - 災害や事故に関する不適切な評論
    - 性的示唆・挑発的内容
    - 血なまぐさい・残虐・恐怖など不快な表現
    - 差別の扇動
    - 低俗・庸俗・媚俗な内容

    - 侮辱、脅迫、嫌がらせ
    - スパム、悪意ある情報、詐欺的情報
    - 個人情報の侵害
    - 肖像権、知的財産権、営業秘密などの侵害
    - その他、運営を妨げる内容
