# 同期
同期を使うと、次のことが可能になります：
- 同じ図書館アカウントで複数端末にログインし、競合を回避
- ほかの機能も順次対応予定

同期機能は適切なタイミングで公開されます。「マイ」->「掲示板」でも最新情報を確認できます。

## Hamアカウント
同期を使うにはHamアカウントへのログインが必要です。HamはQQ、WeChat、Github、自強Auth、Apple（iOS）のログインに対応しています。

ログイン後、プロフィールでログイン済み端末や連携済みアカウントを確認できます。端末の削除や連携解除も可能です。

## 必要な権限
- 通知 - 同期ステータス用
- ネットワーク - Hamサーバー接続と他端末への中継用
- 第三者ログイン - 身元確認用
