# ユーザーセンター

Hamアカウントにログインすると、ユーザーセンターでアカウント情報や設定を管理できます。

## Hamアカウント

Hamは以下のログイン方法に対応しています：
- Passkey
- QQ
- WeChat
- GitHub
- 自強Auth
- Appleサインイン（iOSのみ）

## プロフィール

ログイン後、ユーザーセンターでアバターやニックネームを変更できます。

## ログイン端末

現在ログインしている端末の一覧を確認し、使用しなくなった端末を削除できます。

## ソーシャルアカウント

連携済みのソーシャルアカウントを確認したり、新しいアカウントを連携したり、既存のアカウントの連携を解除したりできます。対応するソーシャルアカウントの種類はログイン方法と同じです。

## Passkey管理

ここでPasskeyを管理できます。新しいPasskeyの追加や既存のPasskeyの削除が可能です。Passkeyはパスワード不要のログイン方法で、端末の生体認証（Face IDや指紋など）を使用して本人確認を行います。

## QRコードログイン

アプリを使ってHam WebのログインQRコードをスキャンし、素早くログインできます。

## 認可済みアプリ

Hamアカウントでログイン認可した第三者アプリを確認・管理できます。使用しなくなったアプリの認可を取り消すことも可能です。

## その他

「その他」では、アカウントの退会などの操作が行えます。
