# 授業
Hamは高度にカスタマイズ可能な時間割を提供します。授業をインポートすると、適切なタイミングで通知します。

## 授業の追加・編集・削除
::: tip 編集や削除の前に、以下の用語を理解してください
- **この授業（1コマ）**

  週内の連続した時間帯にある**単一**の授業

- **同じ時間帯の同一科目**

  同じ時間帯・同じ科目コードの全授業

- **この科目**

  同じ科目コードの全授業


> **例**
>
> 張三さんが高等数学（火曜2-4限、木曜6-8限、1-17週）を履修している場合：
>
> ある週の火曜2-4限 -> 1コマ
>
> 1-17週の火曜2-4限 -> 同じ時間帯の同一科目
>
> 1-17週の火曜2-4限と木曜6-8限 -> 1科目

:::

授業ページでは、次の操作ができます。
### タップ
授業をタップすると詳細を確認できます。

教務システムからインポートした授業の場合、科目の種別や単位を表示します。

Hamの成績分布に登録されている場合、分布情報が表示されます。

授業詳細から予定を追加でき、追加した予定は授業に紐づきます。授業に関連する予定の一覧も確認できます。

### 長押し
空白を長押しすると授業の追加や貼り付けができます。

授業を長押しすると、次の操作ができます。

- **編集**

  同じ時間帯の同一科目の基本情報を変更

- **コピー / 切り取り**

  この1コマのみをコピーまたは切り取り

- **削除**

  選択に応じて削除

### ドラッグ
授業をドラッグして時間を簡単に変更できます。

## 時間割の設定
時間割設定で、学期開始日や背景画像の設定、時間割のインポートができます。

::: tip リマインダー
学期開始日を設定しないと、Hamが正しい学期を認識できず、通知が正確になりません。
:::

背景画像の設定や授業の透明度の変更、~~ガウスぼかしの透明度~~も調整できます。

## ウィジェット
時間割ウィジェットをホーム画面に置いて、今日や明日の授業を素早く確認できます。
