# スポーツ施設

::: tip バージョン要件
iOS: 1.3.2 (build >= 96)

Android: 1.4.3
:::

Hamでスポーツ施設を予約できます。

## 準備
予約前に「マイ」->「情報ポータル設定」で情報ポータルアカウントにログインしてください。

## 施設の閲覧
スポーツ種目と施設を選び、空き枠や時間帯を確認して予約できます。

施設閲覧が向くケース：
- 突然運動したくなったが予約がない
- 施設の空きが十分にある

## お気に入り予約
設定でお気に入りの施設・枠・時間帯を登録できます。登録するとホームに表示され、Siriショートカットで自動化予約も可能です。
::: tip 注意
ホーム画面のクイック予約やSiriショートカットでは、空き状況の確定は行いません。予約に失敗した場合、Hamが近くの空き枠を自動で選びます。
:::

## 注文センター
ホームから注文センターへ移動し、注文の確認やキャンセルができます。

## 支払い
予約が成功し支払い時間になったら、以下の方法で支払えます：
- WeChatミニプログラムで支払う：

「場館预约」ミニプログラム内で支払い。WeChat Payに対応。

- Webで支払う：

確認後、大学の公式決済プラットフォームへのリンクが提供されます。ブラウザに貼り付けて支払いを完了します。AlipayまたはUnionPayに対応。

::: warning 重要
武漢大学の決済プラットフォームでの支払いはHamとは無関係です。
:::

## 予約ステータス
予約状況はホームに表示されます。

支払い時間になると、ステータスカードから支払いができます。

## 自動化
Siriショートカットを使って自動予約できます。
