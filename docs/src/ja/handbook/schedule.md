# 予定
予定ページでは、すべての予定の確認、追加、削除ができます。

## 直近の予定
予定ページに入ると、ページ上部に現在時刻から最も近い未期限の予定が表示されます。予定名、時間、場所、開始までのカウントダウンが含まれます。授業に紐づいている場合は、授業名も表示されます。

直近の予定カードをタップすると、詳細を確認できます。

## 予定グループ
グループを作成して予定を分類管理できます。グループは予定リストの上部に表示され、タップするとそのグループの予定のみが表示されます。各グループには名前とアイコンを設定できます。

## 予定の追加
右上の「+」ボタンをタップして新しい予定を追加します。以下の情報を設定できます：

- **名前**：予定のタイトル（必須）
- **場所**：予定の場所
- **メモ**：予定の詳細内容
- **目標時間**：予定の開始時間
- **終了時間**：任意。予定の終了時間（開始時間から24時間以内）
- **グループ**：予定をグループに分類
- **リマインダー**：予定開始前に通知。10分、30分、1時間、2時間、4時間、8時間から選択可能
- **繰り返し**：日、週、月単位の繰り返しルールを設定可能。繰り返し終了日も設定できます
- **関連授業**：時間割の授業と予定を紐づけ

## 編集と削除
予定リストの項目をタップすると詳細が表示されます。詳細ポップアップでは：
- 設定ボタンをタップして予定を編集
- 削除ボタンをタップして予定を削除
- 繰り返し設定がある場合、関連するすべての繰り返し予定を削除することも可能

## システムカレンダー連携（iOS）
予定の追加・編集後、Hamは自動的にシステムカレンダーに同期します（カレンダー名：「Ham-日程」）。予定名、場所、メモ、時間、リマインダー情報がすべて同期されます。

## 通知リマインダー
リマインダーを設定すると、予定開始前にシステム通知でお知らせします。通知には予定名、場所、開始時間が含まれます。

## 必要な権限
### 予定リマインダー
- iOS: 通知、カレンダー
- Android: 通知権限
