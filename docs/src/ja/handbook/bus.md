# キャンパスバス

::: tip バージョン要件
iOS: 1.3.2 (build >= 96)

Android: 1.4.3
:::

Hamでキャンパスバス情報を確認できます。

## 準備
使用前に「マイ」->「情報ポータル設定」で情報ポータルアカウントにログインしてください。

## ステータス
ステータスページの「キャンパスバス」カードに、最寄り停留所のバス情報が表示され、自動更新されます。

## ダイナミックアイランド / LiveActivity（iOS）
ステータスページのキャンパスバスカードにダイナミックアイランド/LiveActivityを追加できます。この機能はバックグラウンド位置情報の権限を使用します。
