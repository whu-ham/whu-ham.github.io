# 概要

### Hamは安全ですか？
Hamは、学籍番号やパスワードを第三者サーバーへ送信して認証することはありません。アカウント情報は端末内にのみ保存され、端末は大学のサーバーへ直接アクセスします。

一部の機能（成績分布など）ではバックエンドが必要です。データ送信が必要な場合は必ず同意を得て行い、重要な識別情報（学籍番号など）は送信前にハッシュ化します。これにより、運営側が個人を特定することはできません。

### なぜ今はオープンソースではないのですか？
プロジェクトにはプッシュサービスやApple開発者アカウントなどの機密情報が含まれているため、現時点では公開していません。ただし、Hamは完全に無料です。

ご意見はGitterでの交流や技術ディスカッションでお待ちしています。
