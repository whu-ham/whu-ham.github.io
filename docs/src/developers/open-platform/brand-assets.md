## 使用规范

**推荐：**

- 保持 Icon 原始比例，避免拉伸或压缩
- 在 Icon 周围保留适当的留白空间
- 在深色背景上使用时，确保有足够的对比度

**禁止：**

- 修改 Icon 的颜色
- 在 Icon 上叠加文字或其他元素
- 将 Icon 用于暗示 Ham 官方认证或背书
