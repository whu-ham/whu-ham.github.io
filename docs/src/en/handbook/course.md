# Courses
Ham provides a highly customizable timetable. You can import your courses into Ham, and it will remind you at the right time.

## Add, edit, delete courses
::: tip Before editing or deleting, please understand the terms below
- **This class session**

  A **single** class that occupies a continuous time slot within a week

- **All sessions at the same time**

  All sessions with the same time slot and the same course code

- **This course**

  All sessions with the same course code


> **Example**
>
> Zhang San has Advanced Mathematics (Tue periods 2-4, Thu periods 6-8, weeks 1-17).
>
> In a given week, Tue 2-4 -> one class session
>
> Weeks 1-17, Tue 2-4 -> all sessions at the same time
>
> Weeks 1-17, Tue 2-4 and Thu 6-8 -> one course

:::

The course page supports four actions:
### Tap
Tap a class to view details.

If the class is imported from the academic system, Ham shows the course type and credits.

If the class is in Ham's grading system, Ham shows the grading distribution.

You can add schedule items from the class details, and new items will be linked to the course. You can also view all schedule items related to the course here.

### Long press
Long press an empty slot to add or paste a class.

Long press a class to operate on it:

- **Edit**

  Update the basic information for all sessions at the same time

- **Copy / Cut**

  Copy or cut only this class session

- **Delete**

  Delete based on your selection

### Drag
Drag a class to quickly change its time.

## Timetable settings
In the timetable settings page, you can set the semester start date, background image, or import a timetable.

::: tip Reminder
You must set the semester start date so Ham knows which term you are using and can remind you correctly.
:::

You can also set a background image, adjust class transparency, ~~and Gaussian blur transparency~~.

## Widgets
Ham provides timetable widgets. You can place a widget on your home screen to quickly view today's or tomorrow's classes.
