# Shuttle

::: tip Version requirements
iOS: 1.3.2 (build >= 96)

Android: 1.4.3
:::

You can view campus shuttle information with Ham.

## Preparation
Before using the shuttle feature, log in to your information portal account in "Me" -> "Information Portal Settings."

## Status
The "Shuttle" card on the status page shows campus bus information for the nearest stop and updates automatically.

## Dynamic Island / LiveActivity
You can add a Dynamic Island/LiveActivity to the shuttle card on the status page. This feature uses background location permission.
